# Leçon 1 : Introduction à Node.js

## Objectif
Comprendre les avantages et inconvénients de Node.js


### 1. Qu'est-ce que Node.js ?
   - **Node.js en un coup d'œil** : Node.js est une plateforme JavaScript côté serveur qui permet d'exécuter du code JavaScript en dehors du navigateur. Cela ouvre la porte à une variété d'applications, de serveurs web à des outils en ligne de commande.
   - **Pourquoi Node.js ?** : Node.js est populaire pour son efficacité et sa polyvalence, notamment dans le développement de serveurs, d'applications web en temps réel et de microservices.
   - **Qu'est-ce que Node.js n'est pas ?** : Node.js n'est pas un langage de programmation, un framework ou une bibliothèque. C'est une plateforme qui permet d'exécuter du code JavaScript en dehors du navigateur.
